1. This guide assumes you have a Python3 installation because it is written for Python3
2. Open up your favourite terminal application. (Cmd, Powershell, Terminal etc...)
3. Install matplotlib and numpy for Python3 by using the following commands ->
pip3 install matplotlib
pip3 install numpy
4. Run the program by using the following command ->
python linear_classifier.py
5. When asked for the name of the input file then enter -> 'iris.data.txt' and then press enter
